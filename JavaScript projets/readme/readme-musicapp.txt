Projet MusicApp

Le projet consiste à un émettre un son juste en cliquant sur les pads colorés. Less sons se sont des
mini bruitage de 3seconde

Dans le code nous avons pu amélioré qu'en cliquand sur un des pads une bulle apparaisse et se dissimule
derrière le pad. Nous avons aussi par la suite cahngé le css pour l'esthétique.
