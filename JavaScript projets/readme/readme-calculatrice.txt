Voici mon projet calculatrice.

Par rapport au projet de base, le résultat d'un calcul
se stock dans l'input. Il est donc possible de 
faire un calcul avec un résultat obtenu préalablement.

Le bouton "Ce" permet de supprimer le dernier chiffre
Le bouton "C" réinitialise à 0 l'input.

J'ai ajouté le bouton "sqrt" qui permet de faire la 
racinne carré du nombre présent dans l'input. 

Le principe de la calculatrice est d'ajouté un caractère 
à la suite dans l'input puis de faire faire le calcul 
présent dans l'input, le calcul est donc lu.
