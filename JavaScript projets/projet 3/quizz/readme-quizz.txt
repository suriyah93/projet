Le projet quizz.

Ce projet est un quizz avec toutes les questions d'affilée.
Les réponses possible sont soit un input bouton type radio 
soit un input type textarea. 

De base il n'y avait pas de css, j'ai donc centrer le tout 
et mis un background color.
J'ai aussi rajouté un texte avec le nombre de bonne réponse
lors de la validation des réponses. En fonction, 
Un smiley avec un background color différent s'affiche.
