Projet Snake version officiel : snake.html

Le projet se base sur le jeu mythique Snake. Dans le jeu vous incarnez un serpent. Le but du jeu c'est d'essayer 
d'attraper un maximum de pommes sans mourir.Plus(+) le serpent attraper de pomme plus ajouter un bloc de couleur à sa queue.
Du coup le serpent deenait de plus en plus long. On perd uniquement que quand on revient sur nous(manger soi-même).

Pour cela j'ai utlisé le lagage JavaScript.

De base dans le code initial nous avons eu le droit juste au jeu avec un cadre noir avec le serpent et la pomme 
et nous pouvons y se dépacer dans le jeu. 

Par la suite j'ai ajouté un système de score pour connaitre le nombre de fois qu'on attrape une pommes et j'ai
aussi fait en sorte que le résulatse renitialise à chaque fois qu'on perd.
